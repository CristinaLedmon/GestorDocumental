#### Documentación

1. Inicialmente debes tener instalado nodejs y npm.

- [https://nodejs.org/en/download/package-manager](https://nodejs.org/en/download/package-manager)

2. (Opcional) Si deseas instalar pnpm utiliza este comando.

```bash
npm install -g pnpm
```

3. Posteriormente a esto y verificado que disponemos de node, procederemos a arrancar el proyecto en nuestro editor de codigo preferido.

```bash
npm run dev
# o
yarn dev
# o
pnpm dev (recomendable ya que es más rápido)
```

Para acceder al panel en desarrollo: [http://localhost:3000](http://localhost:3000)
